# MMM Checklist Sistemi

Endüstriyel checklist ve görev yönetim sistemi. Kullanıcı rolleri, departman yönetimi, checklist şablonları, görev takibi, onay süreçleri, fotoğraf yükleme sistemi, **envanter yönetimi**, **kalıp seçimi sistemi** ve performans analizi ile kapsamlı bir iş akışı yönetim platformu.

## 🚀 Özellikler

### ✅ Kullanıcı Yönetimi
- Admin tarafından kullanıcı ekleme/düzenleme/silme
- Rol ve departman bazlı yetkilendirme
- Güvenli şifre yönetimi (minimum 6 karakter)
- Çoklu departman ataması

### 🔐 Rol ve Yetki Sistemi
- **9 Modül Bazlı Erişim Kontrolü**:
  - Dashboard
  - Kullanıcı Yönetimi
  - Rol Yönetimi
  - Departman Yönetimi
  - Checklist Yönetimi
  - Görev Yönetimi
  - Kontrol Bekleyenler
  - Performans
  - **Envanter Yönetimi** *(Yeni)*
- Checklist görme/puanlama yetkileri
- Dinamik rol oluşturma ve düzenleme
- Modül bazlı erişim/düzenleme yetkileri

### 🏢 Departman Yönetimi
- Departmanlar arası yetki kontrolü
- Çapraz departman görev görme/puanlama
- Departman bazlı kullanıcı filtreleme

### 📋 Checklist Sistemi
- **Rutin Checklistler**: Günlük, haftalık, aylık periyotlar
- **İşe Bağlı Checklistler**: Kalıp değişim, makine arızası vb.
- **Kalıp Seçimi Sistemi**: Envanterdeki kalıpları seçme ve arama *(Yeni)*
- Dinamik soru ve puanlama sistemi
- Rol bazlı checklist görünürlük kontrolü

### 📦 Envanter Yönetimi Sistemi *(Yeni)*
- **Kategori Yönetimi**: Plastik enjeksiyon makinaları, kalıplar, yardımcı ekipmanlar
- **Dinamik Alan Şablonları**: Her kategori için özelleştirilebilir alanlar
- **Excel Import/Export**: Toplu veri yükleme ve dışa aktarma
- **QR Kod/Barkod Desteği**: Fiziksel takip sistemi
- **Bakım Takibi**: Periyodik bakım planlaması
- **Lokasyon Yönetimi**: Fiziksel konuma göre envanter takibi
- **Veri Kalitesi Skorlama**: Otomatik veri eksikliği analizi
- **Gelişmiş Filtreleme**: Kategori, durum, lokasyon bazlı arama

### 🏭 Kalıp Seçimi ve Yönetim Sistemi *(Yeni)*
- **İndirilen Kalıp Seçimi**: Envanterdeki kalıpları dropdown ile seçme
- **Bağlanan Hamade Seçimi**: Kalıp-hammadde eşleştirmesi
- **Arama Özelliği**: Kalıp kodu ve adına göre arama
- **Autocomplete**: Gerçek zamanlı kalıp önerileri
- **Kalıp Detayları**: Kavite sayısı, malzeme türü, üretim bilgileri

### 📊 Görev Takibi ve Onay Sistemi
- Otomatik ve manuel görev atama
- Periyodik görev oluşturma (Task Scheduler)
- Görev tamamlama ve puanlama
- **Makina bazlı görev ataması**: Envanter entegrasyonu *(Güncellenmiş)*
- **Onay Zinciri Sistemi**:
  - Tamamlanan görevler onay bekler
  - Yetkili kişiler onaylama/reddetme
  - Onay notları ve red nedenleri
  - Görev durumu takibi

### 📸 Fotoğraf Yükleme Sistemi
- **Her checklist maddesi için fotoğraf desteği**
- Mobil cihazlardan doğrudan fotoğraf çekme
- Galeri seçimi ile fotoğraf yükleme
- Base64 formatında güvenli saklama
- Fotoğraf ön izleme ve yakınlaştırma
- Kontrol personeli için fotoğraf görüntüleme
- Dosya boyutu ve tip validasyonu (Max 5MB)
- Responsive fotoğraf görüntüleme

## 📦 Kurulum

### Gereksinimler
- Node.js (v14 veya üzeri)
- MongoDB (yerel veya cloud)
- npm veya yarn

### 1. Projeyi İndirin
```bash
git clone <repository-url>
cd MMM
```

### 2. Backend Kurulumu
```bash
cd backend
npm install
```

### 3. Environment Dosyası
`backend/.env` dosyası oluşturun:
```env
MONGO_URI=mongodb://localhost:27017/mmm-checklist
PORT=5000
JWT_SECRET=your-super-secret-jwt-key-here
NODE_ENV=development
```

### 4. Frontend Kurulumu
```bash
cd frontend
npm install
```

## 🚀 Çalıştırma

### Backend'i Başlatın
```bash
cd backend
npm run start:dev
```
Backend http://localhost:5000 adresinde çalışacak.

### Frontend'i Başlatın
```bash
cd frontend
npm start
```
Frontend http://localhost:3000 adresinde çalışacak.

### Tüm Sistemi Çalıştırın (Önerilen)
```bash
# Ana klasörden (MMM/)
npm run start:dev
```
Bu komut hem backend hem frontend'i aynı anda başlatır.

### Production Build
```bash
# ESLint ile build (önerilen)
cd frontend
npm run build

# ESLint olmadan build (geliştirme aşamasında)
npm run build:frontend:no-lint

# Ana klasörden
npm run build:frontend:no-lint
```

## 🔧 API Endpoints

### Authentication
- `POST /api/auth/login` - Kullanıcı girişi
- `GET /api/auth/me` - Mevcut kullanıcı bilgisi

### Kullanıcılar
- `GET /api/users` - Tüm kullanıcıları listele
- `POST /api/users` - Yeni kullanıcı oluştur
- `PUT /api/users/:id` - Kullanıcı güncelle
- `DELETE /api/users/:id` - Kullanıcı sil

### Roller
- `GET /api/roles` - Tüm rolleri listele
- `GET /api/roles/:id` - Rol detayı
- `POST /api/roles` - Yeni rol oluştur
- `PUT /api/roles/:id` - Rol güncelle
- `DELETE /api/roles/:id` - Rol sil

### Departmanlar
- `GET /api/departments` - Tüm departmanları listele
- `POST /api/departments` - Yeni departman oluştur
- `PUT /api/departments/:id` - Departman güncelle
- `DELETE /api/departments/:id` - Departman sil

### Makinalar
- `GET /api/machines` - Tüm makinaları listele
- `POST /api/machines` - Yeni makina oluştur
- `PUT /api/machines/:id` - Makina güncelle
- `DELETE /api/machines/:id` - Makina sil

### Modüller
- `GET /api/modules` - Tüm modülleri listele

### Checklist Şablonları
- `GET /api/checklists` - Tüm şablonları listele
- `POST /api/checklists` - Yeni şablon oluştur
- `PUT /api/checklists/:id` - Şablon güncelle
- `DELETE /api/checklists/:id` - Şablon sil

### Görevler
- `GET /api/tasks` - Tüm görevleri listele
- `GET /api/tasks/my` - Kullanıcının görevleri
- `GET /api/tasks/control-pending` - Kontrol bekleyen görevler
- `POST /api/tasks` - Yeni görev oluştur
- `PUT /api/tasks/:id/complete` - Görevi tamamla (fotoğraf dahil)
- `PUT /api/tasks/:id/score-items` - Görev maddelerini puanla
- `PUT /api/tasks/:id/approve` - Görevi onayla
- `PUT /api/tasks/:id/reject` - Görevi reddet

### Envanter Yönetimi *(Yeni)*
- `GET /api/inventory/categories` - Envanter kategorileri
- `POST /api/inventory/categories` - Yeni kategori oluştur
- `GET /api/inventory/categories/:id/fields` - Kategori alan şablonları
- `POST /api/inventory/categories/:id/fields` - Yeni alan şablonu
- `GET /api/inventory/items` - Envanter öğeleri (gelişmiş filtreleme)
- `POST /api/inventory/items` - Yeni envanter öğesi
- `PUT /api/inventory/items/:id` - Envanter öğesi güncelle
- `DELETE /api/inventory/items/:id` - Envanter öğesi sil
- `GET /api/inventory/categories/:id/excel-template` - Excel şablonu indir
- `POST /api/inventory/categories/:id/excel-import` - Excel import
- `GET /api/inventory/categories/:id/excel-export` - Excel export
- `GET /api/inventory/machines-for-tasks` - Görevler için makina listesi
- `GET /api/inventory/kalips-for-tasks` - Görevler için kalıp listesi *(Yeni)*
- `GET /api/inventory/dashboard` - Envanter dashboard istatistikleri
